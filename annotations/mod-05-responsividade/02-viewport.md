# Viewport

É uma tag meta que definimos o tamanho da tela com base no tamanho do dispositivo que acessa o nosso conteúdo
normalmente quem desenvolve para mobile usa a propriedade **user-scalable=no** para que não seja possível dar zoon no 
conteúdo da web-page

~~~ html
    <head>
    <meta name="viewport" content="width=device-width, user-scalable=no"  />
    </head>
~~~
